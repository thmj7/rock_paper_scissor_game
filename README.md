# rock_paper_scissor_game
A basic rock-paper-scissor game interface using Python programming.
# Libraries
1. ktinker
2. random
   This is a user vs computer game. We have imported random, so that the computer can choose a random choice from rock, paper, scissors.
Instead of giving rock, paper, and scissors options to the computer. I have created a directory as
-1: "🪨 Rock",
0: "📄 Paper",
1: "✂️ Scissors"
I have created user-defined functions like decide_winner, play, reset game, play sound, and flash animations to make the code more organised and attractive.
